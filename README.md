# Project-school-21

Репозиторий с артефактами бизнес/системного анализа по учебному проекту School 21.
Содержит документацию, диаграммы и модели, которые описывают предметную область, требования и бизнес-процессы.

## Структура
- **BSA00_Decomposition** — декомпозиция системы/функций, scope, верхнеуровневые элементы
- **BSA01_Stakeholders** — стейкхолдеры, роли, ожидания, карта интересов
- **BSA02_Requirements** — требования (functional / non-functional), критерии приемки, приоритеты
- **BSA03_HowToRequirements** — подход/правила описания требований, шаблоны, примеры
- **BSA04_Domains** — доменная модель: сущности, связи, словарь терминов
- **BSA05_Diagrams** — диаграммы (контекст, use case, sequence/flow и др.)
- **BSA06_BPMN** — BPMN-модели бизнес-процессов (AS-IS / TO-BE при наличии)

## Что можно посмотреть в первую очередь
1. Requirements → чтобы понять функционал и ограничения
2. Domains/Diagrams → чтобы увидеть модель и логику системы
3. BPMN → чтобы понять процессы и сценарии

## Статус
В процессе пополнения/актуализации.
